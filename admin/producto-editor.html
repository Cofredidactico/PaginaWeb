🔑 Conectar con GitHub

Para cargar y guardar productos necesitás un Personal Access Token de GitHub.

Cómo generarlo (solo una vez):
1. Andá a github.com/settings/tokens →
2. Poné cualquier nombre · Expiration: No expiration
3. Tildá solo repo · Clic en "Generate token"
4. Copiá el token (empieza con ghp_) y pegalo acá

🔒 Se guarda solo en este navegador. Nunca sale de tu computadora.
