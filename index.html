Configuración Firebase

Conectá tu base de datos

Necesitás un proyecto Firebase con Firestore activo. Solo se configura una vez.

Antes de continuar, verificá en Firebase Console:
1. Firestore Database → Crear base de datos → Modo Prueba → nam5
2. Firestore → pestaña Reglas → publicá esto exactamente:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
¿Dónde está el firebaseConfig?
Firebase Console → ⚙️ Configuración del proyecto → General → Tus apps → </> Web
Paso 2 de 3

Ingresá tu firebaseConfig

Copiá los valores del objeto firebaseConfig de tu proyecto.

Después de completar el pago en la plataforma (Empretienda o Hotmart), recibís un email con el link de descarga. Para los materiales gratuitos, dejás tu email en el formulario y te lo enviamos al instante.

Sí, todos los materiales están diseñados para imprimir en impresora doméstica o en una copistería. Recomendamos impresión en color para aprovechar al máximo los diseños, aunque muchos también funcionan en blanco y negro.

Primero revisá la carpeta de spam o correo no deseado. Si no lo encontrás en 30 minutos, escribinos a cofredidacticoo@gmail.com con el comprobante de pago y te lo enviamos de inmediato.

Los materiales son para uso personal y educativo. No está permitida la reventa, distribución masiva ni modificación con fines comerciales. Podés usarlos en tus clases, con tus alumnos, y compartir el link de la tienda con colegas.

La plataforma de pago (Empretienda/Hotmart) genera automáticamente un comprobante al completar la compra. Si necesitás una factura específica, escribinos a cofredidacticoo@gmail.com.
