Configuración Firebase

Conectá tu base de datos

Necesitás un proyecto Firebase con Firestore activo. Solo se configura una vez.

Antes de continuar, verificá en Firebase Console:
1. Firestore Database → Crear base de datos → Modo Prueba → nam5
2. Firestore → pestaña Reglas → publicá esto exactamente:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
¿Dónde está el firebaseConfig?
Firebase Console → ⚙️ Configuración del proyecto → General → Tus apps → </> Web
Paso 2 de 3

Ingresá tu firebaseConfig

Copiá los valores del objeto firebaseConfig de tu proyecto.

Paso 3 de 3

Elegí tu contraseña

Esta contraseña la usás para entrar al panel de admin. Las docentes no necesitan contraseña para ver los materiales.
